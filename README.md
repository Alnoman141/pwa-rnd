# vue-pwa-camera
A simple camera progressive web app built with Vue.js.

## [Live Demo :rocket:](https://vue-pwa-camera.netlify.app/)

## Project setup
```bash
npm install
```

### Compiles and hot-reloads for development
```bash
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
